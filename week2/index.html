Variables: Valuables are used to refer to a value stored in memory
Declaring and assigning:
Variables have to be declared before they can be used. From ES6 onwards, JavaScript uses the keywords const and let to declare variables. The keyword const is used when the variable will not be reassigned to another value.
WHEREAS The keyword let is used if the variable might be reassigned later in the program. TO assign a value to a constant or variable, we use the = operator.
