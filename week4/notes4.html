Object-oriented Programming

• OOP is a method of separating the code into objects with properties and methods that you call.

• There are many different things you can do in OOP(object-oriented programming), such as encapsulation, polymorphism, and more.
• Encapsulation is just exposing only important parts of programming like an on button.
•Polymorphism is like taking a class and morphing it for example to do something else or add to it. for example, you could have the animal class which extends to the dog class
